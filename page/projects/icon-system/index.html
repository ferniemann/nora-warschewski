Icon System

The icon system focuses on illustrative icons depicting topics from the field of energy, environment, and safety.

Conceptionally, it is a modular system with reusable components and proportional. It is intended to snap to the grid in design software to reduce aliasing.

Currently, the project is in the stage where all icons get sketched out and grouped according to topic. In the next step, I will clean up paths and create the typeface. Ultimately, the system is supposed to be published as an expandable OpenType typeface with multiple styles.

Early stage: Grouping of subject groups.

Early stage: Grid system, guides, and basic shapes as a foundation to build components.
